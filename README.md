## üìñ Overview
**CL Meter** is a lightweight, fully customizable network speed monitor designed for especially Windows 11 as Microsoft removed the native "Taskbar Toolbars" API which restricts alternative speed meter intigration in the taskbar. It floats seamlessly on your taskbar, providing real-time Download and Upload speeds without cluttering your workspace.

Unlike other paid alternatives, CL Meter is completely **free to use**, privacy-focused, and packed with advanced tracking features like historical data usage and daily reports.

## ‚öôÔ∏è How to Use
* **Default:** By default, the CL Meter speed indicator will appear in the right-bottom task bar.
* **Move It:** If you want to change position, click and drag the widget to position it anywhere on your taskbar.
* **Menu:** Right-click the widget (or the System Tray icon) to open the menu.
* **Lock:** Lock the position from menu or settings.
* **Settings:** Go to `Settings` to change colors, size, download-upload indicators, and lock the position.
* **Reports:** Click `Reports` to see your data usage history and graphs.
* **Exit:** Right-click the widget (or the System Tray icon) and select `Exit`.
